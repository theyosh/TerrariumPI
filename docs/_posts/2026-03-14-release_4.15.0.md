## Release 4.15.0 - Spring release: More hardware

In this release we add some more hardware devices. 2 Relays and 1 new sensor are
added.

  IoT]({% link _hardware/16-channel_relay.md %}) \\

### Sensors

- [AS7331 UVA, UVB, UVC light and temperature

### Other updates

- Added PCA9685 dimmer calibration
- Updates all NodeJS, Python and Ruby libraries for the GUI, TerrariumPI and the
  documentation.
- Updated the documentation at various pages. Mostly adding more information or
  corrections.
- Translations are updated
- Refactored the [database cleanup]({% link _faq/clean_database.md %}) code
- Updated copyright date
